Custom Minecraft plugin pack for **Paper 1.21+** servers.  
Made by **xxenn / ra1nzao**.

This repository contains custom server plugins designed to be easier to use, simpler to manage, and cleaner to run than many large server plugin packs.

## Included Plugins

### Ra1nCore
A large utility/core plugin for Paper servers.

Main features:
- homes
- warps
- spawn
- tpa / tpahere / tpauto
- fly
- rtp
- economy
- shop
- sell
- sell history
- crates
- team system
- stats / leaderboard
- clearlag
- combat tools
- portals
- phantom control
- spawner tools
- player tracker
- item control
- mythic-style tools
- admin GUIs
- yellow themed messages and inventories

### Ra1nOrder
A custom order system plugin for Paper servers.

Main features:
- public orders GUI
- your orders GUI
- admin orders GUI
- create orders from held items
- deliver items into open orders
- collect stored delivered items
- search / filter / sort
- pagination
- order expiry
- admin tools
- order logs
- webhook support
- yellow themed inventories
- no license system
- no DRM

## Why This Pack

This pack was made to provide:
- simpler usage
- custom branding
- easier management
- yellow themed style
- no forced license systems
- no DRM
- no hidden premium lock features
